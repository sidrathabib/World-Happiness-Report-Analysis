# 🌍 Global Happiness Data Analysis

## 📌 Overview

This project explores the World Happiness Report dataset to identify key factors that influence happiness across countries. Using data analysis and visualization techniques, the project examines how economic, social, and health-related variables relate to overall life satisfaction.

---

## Objectives

* Identify the happiest countries in the world
* Analyze relationships between happiness and key factors such as GDP and social support
* Explore correlations between multiple variables
* Generate insights into global patterns of wellbeing

---

## Tools & Technologies

* Python
* pandas
* seaborn
* matplotlib
* Jupyter Notebook

---

## Key Analyses

### 1. Top 10 Happiest Countries

* Identified countries with the highest happiness scores
* Visualized results using a bar chart

### 2. Correlation Analysis

* Computed correlations between numerical variables
* Visualized relationships using a heatmap

### 3. GDP vs Happiness

* Explored the relationship between economic output and happiness
* Found a positive association between GDP and life satisfaction

### 4. Social Support vs Happiness

* Analyzed how social connections influence happiness
* Observed a strong positive relationship

---

## Key Insights

* Countries with higher GDP per capita generally report higher happiness levels
* Social support is one of the strongest predictors of happiness
* Health and life expectancy are closely associated with life satisfaction
* Happiness is influenced by a combination of economic and social factors
